# Market Basket Analysis – J&K Bank Customer Behavior

## Overview
Analyzed survey responses from 400 J&K Bank customers to identify 
product usage patterns and cross-sell opportunities using the 
Apriori algorithm.

## Tools Used
Python | Pandas | mlxtend | Apriori Algorithm

## Key Findings
- Mobile banking + savings account → app satisfaction 
  [Confidence: 82.46%, Lift: 1.60]
- Credit card + loan → fixed deposit likelihood 
  [Confidence: 81.63%, Lift: 1.48]
- Multi-product customers → likely to recommend bank 
  [Confidence: 80.76%, Lift: 1.50]

## Business Impact
Delivered cross-sell recommendations and referral incentive 
program strategy to bank management.
